# Cricket Match Simulator

A console-based Java application that simulates a cricket match between two teams. The project was developed to practice Object-Oriented Programming concepts and implement real-world cricket scoring logic.

### Features

* Multiple match types (ODI, T20, Test, Custom Match)
* Toss simulation
* Batting and bowling management
* Wicket handling
* Wide ball and no-ball support
* Free-hit implementation
* Over-wise statistics
* Match result calculation

### Technologies Used

* Java
* OOP (Inheritance, Polymorphism, Abstraction, Encapsulation)
* Interfaces
* Exception Handling

### Run the Project

```bash
javac custexc/*.java interfaces/*.java player/*.java team/*.java match/*.java
java match.CricketMatch
```
